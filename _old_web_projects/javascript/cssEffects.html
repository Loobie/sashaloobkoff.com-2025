This text is positioned absolutely using stylesheets. The upper-left corner of the invisible box that surrounds this paragraph is positioned exactly 100 pixels from the all edges of the browser window (top, right, left, bottom). Cool. Actually, the CSS rule is applied to a tag, which in turn surrounds the tag in the HTML. This is necessary because IE has problems positioning elements directly. But if you wrap an element in a and then position the, the element will be positioned fine. That's an excellent general rule to follow when positioning anything.
